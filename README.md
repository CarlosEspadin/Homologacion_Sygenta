# Homologacion_Sygenta
Para actualizar el catalogo de productos vigentes solo hay que sustituir el archivo Precios_Facturación_CP.xlsx por la versión más reciente. Se debe respetar el mismo foramto de columnas SKU y Producto para que funcione correctamente.

Se sugiere que para ejecutar correctamente el programa lo hagan como administradores para evitar.

Les recomendamos respetar el orden de entrada de los catalogos a homologar para mejorar la experiencia de usuario.
